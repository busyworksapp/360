About 2FA
What is Two-Factor Authentication?

2FA adds an extra layer of security by requiring both your password and a time-based code from your phone to log in.

Why should I enable it?
  • Protects against password theft
  • Prevents unauthorized access
  • Industry best practice for admin accounts
  • Required for PCI compliance
How does it work?
  1. Enter your password as usual
  2. Open your authenticator app
  3. Enter the 6-digit code shown
  4. You're logged in!
Important: Don't lose access to your authenticator app! Make sure to back it up or save backup codes.
